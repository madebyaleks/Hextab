Using Hextab in Safari

Apple recently ended the free Safari Developer Program which means that it costs me $99/year to publish Hextab at the Extension Gallery. You can still use Hextab in Safari, it's just a little different setup process:

  1. Open Safari settings by pressing 'cmd + ,'
  2. Set New tab opens with to Homepage
  3. Set Homepage to http://simmple-lab.s3.amazonaws.com/hextab/index.html
